# cs50web
CS50 edx python and javascript

Web Programming with Python and JavaScript

Simple web pages showing some information about a Dungeons and Dragons player character.

	index.html - contains links to other pages with character data
	attributes.html - lists the 6 basic character ability scores and modifiers
	background.html - lists the background information for the character
	equipment.html - lists the current equipment inventory of the character
	character.html - shows an image of the character and overview information
